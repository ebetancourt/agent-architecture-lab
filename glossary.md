# Glossary

## Agent Loop

The recurring cycle by which an agent observes state, decides next action, acts, and evaluates results.

## Context Engineering

The practice of selecting, shaping, compressing, and ordering information supplied to an agent.

## Memory

Stored information reused across turns, sessions, tasks, or projects.

## Tool Use

Structured interaction between an agent and external capabilities such as shell commands, APIs, browsers, or file systems.

## Runtime Architecture

The execution environment, orchestration model, permissions, and state management used by an agentic system.

## Agent UX

The user experience of working with an agent, including approvals, progress updates, corrections, and review.

## Multi-Agent System

A system where multiple agents collaborate, delegate, critique, or run tasks in parallel.
