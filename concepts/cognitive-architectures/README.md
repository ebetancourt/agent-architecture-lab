# Cognitive Architectures

## Definition

Cognitive architecture describes the organization of reasoning roles, memory, perception, action, reflection, and control inside an agent system.

## Architectural Notes

- In software agents, cognitive structure is often distributed across prompts, runtime policy, tools, files, and UI.
- Reflection, planning, and memory should be studied as mechanisms, not labels.
- The useful question is which structure improves observable behavior under real constraints.
