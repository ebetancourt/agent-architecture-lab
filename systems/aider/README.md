# Aider

## Overview

Aider is a terminal-first coding agent useful for studying chat-mediated edit loops, repository context selection, git-aware workflows, and command feedback.

## Architecture Angles

- How selected files become working context.
- How generated edits are parsed, applied, and repaired.
- How git state shapes safety and review.
- How command output feeds subsequent turns.
