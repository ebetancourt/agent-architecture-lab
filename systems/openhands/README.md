# OpenHands

## Overview

OpenHands is a software-agent runtime useful for studying evented execution, sandbox boundaries, UI-mediated control, and tool orchestration.

## Architecture Angles

- Event representation for actions and observations.
- Runtime and sandbox lifecycle.
- Frontend/backend responsibility split.
- Completion, retry, and user-control surfaces.
