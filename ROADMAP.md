# Roadmap

## 12-Month Progression

| Month | Focus | Outcome |
| --- | --- | --- |
| 1 | Agent loops and harness basics | Understand observe-plan-act-evaluate cycles. |
| 2 | Context engineering | Design prompt, file, and task context strategies. |
| 3 | Memory and state | Compare session, project, and long-term memory patterns. |
| 4 | Tool use | Study tool schemas, execution, validation, and errors. |
| 5 | Runtime architecture | Map process, sandbox, workspace, and orchestration models. |
| 6 | Pi deep dive | Build detailed notes on Pi-style agent runtime design. |
| 7 | Coding-agent UX | Study review, approval, interruption, and collaboration flows. |
| 8 | Multi-agent systems | Compare delegation, parallelism, and coordination patterns. |
| 9 | Evaluation | Define task suites, traces, rubrics, and regression checks. |
| 10 | Security and trust | Review permissions, secrets, sandboxing, and auditability. |
| 11 | Project implementation | Implement selected curriculum exercises. |
| 12 | Synthesis | Produce final architecture review and learning retrospective. |

## Milestones

- Month 3: First architecture comparison notes.
- Month 6: Completed Pi deep dive.
- Month 9: Evaluation plan drafted.
- Month 12: Final curriculum retrospective.
